# TDDE24 - Funktionell och imperativ programmering, del 2

My lab solutions for the second part of the introductory programming sequence (TDDE24) at Linköping University. This course continues directly after TDDE23.

## Course Structure
The course covers labs 6-8, with a focus on more advanced topics:

- **Lab 6** - Languages and interpreters (done in pairs)
- **Lab 7** - Algorithms
- **Lab 8** - Data abstraction (larger assignment)

## Repository Structure
```
├── lab6/               # Languages and interpreters
├── lab7/               # Algorithms
├── lab8/               # Data abstraction (calendar project)
├── documentation/      # Personal notes, todo.md, etc.
├── inför_tentamen/     # Exam preparation materials
└── add-upstream-remote.sh  # Helper script for git setup
```

### Lab Details
- **Lab 6**: Working with interpreters and language implementation (`calc.py`, `upg6.py`)
- **Lab 7**: Algorithm exercises with books and matching problems
- **Lab 8**: Larger project building a calendar application with multiple modules focusing on data abstraction

## About the Course
TDDE24 is the second part of the introductory programming course for Computer Science students at LiU. It covers algorithms, data abstraction, and more complex programming projects.
